# Лабораторная работа 2

## Инженерная задача
1. Установить Ollama.
2. Скачать модель Qwen2.5:0.5B.
3. Поднять Ollama сервер с LLM.
4. Написать скрипт, отправляющий запросы на Ollama сервер.
5. Прогнать скрипт через 10 произвольных запросов и зафиксировать вывод LLM.
6. Составить отчет инференса с двумя столбцами: запрос и соответствующий вывод.

## Использованный стек
- Ollama
- Qwen2.5:0.5B (модель `qwen2.5:0.5b`)
- Python 3.14
- requests

## Как запускать проект

### 1) Установить Ollama
Скачать и установить с официального сайта: https://ollama.com/download

Проверка установки:

```powershell
ollama --version
```

### 2) Скачать модель

```powershell
ollama pull qwen2.5:0.5b
```

### 3) Запустить Ollama сервер
В отдельном терминале:

```powershell
ollama serve
```

### 4) Установить Python-зависимость
```powershell
pip install -r requirements.txt
```

### 5) Запустить скрипт инференса

```powershell
ollama_inference_requests.py
```

После запуска создается/обновляется файл `inference_report.md`.
